# Car Insurance System - Use Case Diagram

## Overview
This diagram represents a Car Insurance System and shows how users interact with it.

## Actors
- Customer
- Insurance Agent

## Main Functions
- Request Insurance Quote
- Buy Insurance Policy
- Verify Driver Information
- Submit Claim
- Track Claim Status
- Review Claim

The customer can request a quote, buy an insurance policy, submit a claim, and track its status. When buying a policy, the system verifies the driver information.

The insurance agent is responsible for reviewing the claims submitted by customers.
